# AuraWatch Premium — Landing Page de Alta Performance

[![GitHub license](https://img.shields.io/github/license/TFS-Data/pagina-de-vendas)](https://github.com/TFS-Data/pagina-de-vendas/blob/main/LICENSE)
[![GitHub stars](https://img.shields.io/github/stars/TFS-Data/pagina-de-vendas)](https://github.com/TFS-Data/pagina-de-vendas/stargazers)

Uma landing page premium, moderna e responsiva desenvolvida para a venda do smartwatch **AuraWatch Premium**. O projeto foca em alta conversão, design elegante com tema escuro e performance otimizada.

## 🚀 Tecnologias Utilizadas

- **HTML5 Semântico**: Estrutura otimizada para SEO e acessibilidade.
- **Tailwind CSS**: Estilização moderna via JIT (Just-In-Time) com sistema de design customizado.
- **JavaScript Vanilla**: Lógica de interação e animações sem dependências pesadas.
- **AOS (Animate On Scroll)**: Animações suaves de entrada durante a navegação.
- **Google Fonts & Material Symbols**: Tipografia e ícones premium.

## ✨ Funcionalidades

- **Design Dark Mode Premium**: Estética baseada no sistema de design *Kinetic Horizon*.
- **Responsividade Total**: Experiência fluida em dispositivos mobile, tablets e desktop.
- **Otimização de Conversão (CRO)**: CTAs estratégicos e hierarquia visual focada no produto.
- **Performance**: Carregamento rápido utilizando recursos leves e assets otimizados.
- **Seção de Saúde e Conectividade**: Layout em *Bento Grid* para destaque de funcionalidades.

## 📁 Estrutura do Projeto

```text
├── assets/
│   ├── css/        # Estilos customizados e overrides
│   ├── images/     # Assets de imagem do produto (Hero, Lifestyle, Flatlay)
│   └── js/         # Scripts de comportamento da página
├── index.html      # Página principal
└── .gitignore      # Configurações de versionamento
```

## 🛠️ Como Executar o Projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/TFS-Data/pagina-de-vendas.git
   ```
2. Abra o arquivo `index.html` em seu navegador ou utilize uma extensão como Live Server.
